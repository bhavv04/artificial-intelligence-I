# CPS721 - Artificial Intelligence

Assignments covering core AI concepts, from knowledge representation 
in Prolog to planning and natural language processing.

## Assignments

### A1 - prolog-foundations
Building knowledge bases in Prolog using facts and rules. Covered movie 
and grocery domains, and recursive path-finding with Six Degrees of Kevin Bacon.

### A2 - list-algorithms
Prolog list manipulation, recursive rules, and graph traversal. Covered 
list equality, power sums, tree paths, and an extended Kevin Bacon problem 
using lists to avoid cycles.

### A3 - constraint-solving
Solving constraint satisfaction problems (CSPs) using generate-and-test 
and interleaving. Covered a delivery truck scheduling problem, a network 
logic puzzle, and KenKen.

### A4 - natural-language
Building a natural language question-answering system in Prolog for a 
movie database. Covered lexicon design, noun phrase parsing, and 
custom parser extensions.

### A5 - ai-planning
Implementing planning systems using the Situation Calculus. Covered 
multi-agent pathfinding and a robot dishwashing problem, with precondition 
axioms, successor-state axioms, and declarative heuristics.
